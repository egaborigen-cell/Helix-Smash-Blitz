# **App Name**: HelixSmash

## Core Features:

- 3D Game Environment: Initialize and manage the Three.js scene, camera, lighting, and renderer for a dynamic 3D game world.
- Bouncing Ball Physics: Implement realistic physics for the ball's descent, bouncing mechanics, and interaction with platforms.
- Rotating Tower & Platforms: Procedurally generate a cylindrical tower with various platforms, including destructible sections and gaps, and implement continuous rotation.
- Collision Detection & Scoring: Detect ball collisions with platforms, trigger platform breakage for destructible types, and update player scores based on successful descents.
- User Input Control: Allow players to rotate the tower using intuitive mouse or touch drag controls to guide the ball through gaps.
- Game State Management: Manage the game's flow, including starting a new game, handling 'game over' conditions when the ball falls off, and offering restart options.

## Style Guidelines:

- Primary color: A vibrant, energetic lime green (#B8F53D) to convey excitement and freshness.
- Background color: A subtle, desaturated green-yellow (#DBE0D1) to provide a light, clean canvas that allows game elements to stand out.
- Accent color: A bold, contrasting golden yellow (#F2CC0D) for highlights, scores, and interactive elements, adding visual pop.
- Headline and body font: 'Poppins' (sans-serif) for its modern, precise, and highly legible appearance, suitable for on-screen scores and instructions.
- Use minimalist and geometric icons for UI elements, ensuring clarity and aligning with the hyper-casual game aesthetic.
- A clean and focused layout that prioritizes the 3D game view, with score and controls minimally overlayed.
- Incorporate satisfying visual feedback animations for ball bounces, platform breaks, and game state changes (e.g., 'Game Over' screen transitions).
